## FirstWayToDo方案介绍
### 问题定义
　　这道题目需要根据一定的骑行信息预测骑行目的地区块，骑行区块由经过加密的字符串表示。由于有太多的区块，用常规的分类方法不好解决；尝试过将字符串解密为经纬度，然后对经纬度进行回归拟合，但是效果不理想。这里借鉴的是推荐的思想，先根据一定的逻辑筛选出候选目的地集合，然后将xgboost的目标函数设置为"rank:pairwise"(用于排序任务)来训练，最终选取预测结果中概率最大的三个目的地。
